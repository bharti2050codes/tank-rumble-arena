# Tank Rumble Arena

A thrilling tank battle game built with React and TypeScript. Control your tank with WASD keys, aim with your mouse, and defeat the AI opponent in intense combat!

## Features

Real-time tank battles
Mouse aiming and shooting
AI opponent with smart movement
Particle effects and explosions
Responsive design

## Controls

**Movement**: WASD or Arrow keys
**Aim**: Mouse
**Shoot**: Left click or Spacebar

## How to Play

Navigate your green tank around the battlefield, avoid obstacles, and take down the red AI tank. First to zero HP loses!

## Development

This project is built with Vite, React, and TypeScript. To run locally:

bash
npm install
npm run dev
